Dodavanje CSS-a u HTML

Što je CSS?
CSS služi za oblikovanje HTML elemenata (boje, fontovi, raspored...)

Načini dodavanja:

  • Inline – unutar taga pomoću style
  • Internal – u style dijelu
  • External – u posebnoj .css datoteci
Preporučuje se koristiti external CSS jer se isti stil može koristiti na više stranica.

Ključni elementi koje smo učili

  • <html> – glavni kontejner za sve HTML elemente
  • <head> – sadrži meta informacije o stranici
  • <body> – sadrži vidljivi sadržaj stranice
  • h1 - h6 – naslovi različitih veličina
  • p – paragrafi teksta
Važno: HTML tagovi se uvijek zatvaraju! Svaki otvoreni tag mora imati svoj zatvoreni tag (osim self-closing tagova poput <br>, <img>).

CSS Opacity

Što je opacity?
Svojstvo opacity određuje prozirnost elementa. Vrijednosti idu od 0 (potpuno nevidljivo) do 1 (potpuno vidljivo).

Napomena: Kada koristiš opacity na elementu, svi njegovi unutarnji elementi (tekst, slike...) također postaju prozirni. Ako želiš da samo pozadina bude prozirna, koristi rgba() boje.

HTML Forme

Što su forme?
HTML forme služe za unos podataka od strane korisnika (ime, email, poruke...).

Najčešći elementi formi:

  • <form> – glavni kontejner forme
  • <label> – opis input polja
  • <input> – unos podataka
  • <textarea> – unos duljeg teksta
  • <button> – gumb za slanje

Napomena: Forme se često povezuju s backendom ili JavaScriptom kako bi se podaci obradili.

CSS Pseudoklase

Što su pseudoklase?
Pseudoklase definiraju posebno stanje elementa (npr. kada miš prelazi preko elementa).

Najčešće pseudoklase:

  • :hover – aktivira se kada miš prelazi preko elementa
  • :focus – kada je element aktivan (npr. input polje)
  • :first-child – prvi element unutar roditelja
  • :last-child – zadnji element unutar roditelja
  • Prvi element
  • Srednji element
  • Zadnji element
Napomena: Pseudoklase omogućuju interakciju bez JavaScripta.

CSS Flexbox

Što je Flexbox?
Flexbox služi za jednostavno poravnanje elemenata u red ili stupac.

Važna svojstva:

  • display: flex
  • justify-content – horizontalno poravnanje
  • align-items – vertikalno poravnanje
  • flex-direction – red ili stupac

CSS Grid

Što je Grid?
CSS Grid omogućuje raspored elemenata u redove i stupce (2D layout sustav).

Važna svojstva:

  • display: grid
  • grid-template-columns
  • grid-template-rows
  • gap – razmak između elemenata
